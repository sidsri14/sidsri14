## Featured Projects

### [StackIntercept](https://github.com/sidsri14/stack-intercept)

Self-hosted Rust proxy for OpenAI-compatible APIs with exact and semantic caching, reactive failover, tenant isolation, Prometheus metrics, Docker deployment, CI, releases, and integration tests.

### [DriftGuard](https://github.com/sidsri14/driftguard)

Released local-first Rust CLI that detects undocumented environment variables and gates changed AI prompts against JSON Schema contracts. Includes Git-diff scoping, reports, diagnostics, pre-commit hooks, 26 automated tests, and cross-platform release binaries.

### [Argus-HFT-Core](https://github.com/sidsri14/Argus-HFT-Core)

Rust low-latency trading infrastructure prototype exploring deterministic execution, zero-copy ingestion concepts, CPU pinning, execution/audit core separation, and HFT-oriented benchmarking.

### PayRecover / Stripeflow

Failed-payment recovery SaaS with Razorpay/Stripe webhooks, BullMQ retry workers, recovery links, auth, CSRF protection, API-key access, audit logs, queue stats, and health checks.

### [Reliable Mail](https://github.com/sidsri14/reliable-mail)

Transactional email API focused on idempotency, retry handling, provider abstraction, API-key auth, append-only usage accounting, and duplicate prevention.
